# tree_exercise

## Purpose:
Sort Binary Tree by Tree Level with Detailed Logging

### Source of the Exercise:
http://www.codewars.com/kata/sort-binary-tree-by-levels/train/ruby

## Binary Tree 1:

                 2
            8        9
          1  3     4   5
          
### Expected result for Tree 1:

[2,8,9,1,3,4,5]

## Binary Tree 2:

                 1
            8        4
              3        5
                         7

### Expected result for Tree 2:

[1,8,4,3,5,7]
